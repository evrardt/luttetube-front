Grâce à vos compétences

Vous êtes Graphiste / WebDesigner / Developpeur Angular / Node JS ? Ça nous intéresse ! Contactez nous par email : enlutte.tube@gmail.com

LutteTube est un projet Open-Source utilisant GitHub

Un simple fork vous permettra de modifier vos fichiers et de soumettre une évolution (graphique, fonctionnalité, améliorations, correction de bug, ...), mais aussi de publier les erreurs rencontrés afin que les développeurs puissent y remédier.

Financièrement

Parce-que l'hébergement d'un site web et la reservation d'un nom de domaine à un coût, LutteTube vous propose de contribuer aux finances grâce à l'envoi d'argent sur notre compte paypal.

Actuellement LutteTube est hébergé chez OVH sur un VPS ce qui permet à l'application de tourner sans restriction.

Suivant le nombre d'utilisateurs simultanés cela peut ne pas être suffisant.
